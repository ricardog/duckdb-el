# emacs-duckdb

An Emacs dynamic module for DuckDB.

## Features
- Performance: Columnar C API for fast data processing.
- Stability: Zero memory leaks with `user_ptr` finalizers.
- Ergonomics: Native Elisp API and `vtable` integration.
